\textbf{Unified metadata schema}
\newline

The unified semantic metadata schema to power reuse of NGS data in research and healthcare. Version 0.1-SNAPSHOT, 2022-02-23. This model consists of 8 modules that contain 121 metadata elements and 100042 lookups in total (excluding null flavors).
